Positioning Parent

This is positioned Static This is the normal flow of elements on a page

this is position : relative; left: 20px This is in the normal flow but we can move it relative to where it should have been

This is position: fixed This is out of the normal flow and scrolls with the page

This is position: absolute; This is out of the normal flow and is positioned in relation to the most immediate non-static parent. Does not scroll with the page

This is position: sticky; top: 50px This is in the normal flow but scrolls with the page as long as it is in it's parent element
